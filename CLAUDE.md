# Claude Instructions

## Project Plan

Always refer to `projectplan.md` for the current state of the project.

- When a task or phase item is completed, mark it as done in `projectplan.md` (check the checkbox and update the status table).
- When the plan changes — new features, dropped features, revised scope, phase restructuring — update `projectplan.md` to reflect the new state.
- Keep the **Next Actions** section current: remove completed items and add new ones as they emerge.
